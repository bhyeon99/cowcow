## 👀 서비스 소개
* 서비스명 : cowcow
* 서비스설명: 실시간 스트리밍 및 가격 가격예측 기반 스마트 경매 플랫폼
<br>

## 📅 프로젝트 기간
2024.10.14 ~ 2024.11.13 (2달)
<br>

## ⭐ 주요 기능
* [기능1] 회원가입 / 로그인 / 소셜 로그인(kakao)
* [기능2] 개인정보수정 / 소등록 / 경매내역 / 회원 탈퇴
* [기능3] 실시간 스트리밍 / 가격 예측
* [기능4] 실시간 경매 / 실시간 알림
<br>
